# Вариант 13 — Микро-шоп «Добавь в корзину» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Пользователь: регистрация → подтверждение → вход → каталог товаров
   - Администратор: вход → админ-панель

2. Просмотр каталога (Пользователь)

   - Просмотр списка товаров с фильтрами (категория, цена, поиск)
   - Просмотр деталей товара (описание, цена, наличие)
   - Добавление товара в корзину

3. Работа с корзиной (Пользователь)

   - Просмотр корзины
   - Изменение количества товаров
   - Удаление товаров из корзины
   - Применение купона на скидку

4. Оформление заказа (Пользователь)

   - Выбор/добавление адреса доставки
   - Применение купона (если есть)
   - Подтверждение заказа
   - Оплата (заглушка) → создание заказа

5. Управление заказами (Пользователь)

   - Просмотр истории заказов
   - Просмотр деталей заказа
   - Отслеживание статуса заказа

6. Администрирование (Администратор)

   - Управление товарами (CRUD)
   - Управление заказами (просмотр, изменение статуса)
   - Управление купонами (создание, редактирование, деактивация)
   - Управление пользователями (просмотр, блокировка)

7. Поиск и фильтрация (Пользователь)

   - Поиск товаров по названию/описанию
   - Фильтрация по категориям
   - Фильтрация по диапазону цен
   - Сортировка (цена, популярность, новизна)
